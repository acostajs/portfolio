# Milestone: v0.2.4 - Advanced Observability & Intelligent Context

## Status: Completed ✅

**Date:** 2026-05-18
**Focus:** Enhancing system visibility, improving chatbot session intelligence, and providing administrative insights.

---

## 🏗️ Infrastructure & CI/CD

### 1. Real-time Visitor Tracking

- **Goal:** Capture anonymized usage data without impacting performance.
- **Task:** Implement a global FastAPI middleware that tracks `Path`, `Locale`, `User-Agent`, and `Referrer`.
- **Strategy:** Use `BackgroundTasks` to persist logs to a new `VisitorLog` table.
- **Security:** Ensure IP addresses are salted/hashed or only stored as approximate geolocation (City/Country).

### 2. Health Check 2.0 (`/health/details`)

- **Goal:** Provide a deep-dive status of the backend ecosystem.
- **Task:** New endpoint returning JSON status for:
  - Database connectivity.
  - Redis/Trigger Cache status.
  - Telegram Bot API reachability.
  - Server resources (Disk/Memory trends).

---

## 🤖 Chatbot & Intelligence

### 3. Contextual Metadata Persistence

- **Goal:** Allow the assistant to "remember" transient session facts.
- **Task:** Add a `metadata: JSON` column to the `LiveChatSession` model.
- **Use Case:** Store detected user intent, preferred language, and "visited_pages" to tailor responses.

### 4. Proactive NLU Hints

- **Goal:** Reduce user friction and guide interaction.
- **Task:** `/api/v1/chat/hints` endpoint that returns 3 suggested questions.
- **Logic:** Map `PageId` to relevant triggers (e.g., if on `/projects`, suggest "What tech did you use for X?").

---

## 🎨 Frontend & DX

### 5. Admin Analytics Dashboard

- **Goal:** Visualize the data collected in Task #1.
- **Task:** Create a dedicated "Stats" tab in the Admin UI.
- **Visuals:** Use simple CSS-based bar charts or a lightweight library (Recharts) to show:
  - Daily active sessions.
  - Most triggered NLU modules.
  - Thumbs up/down feedback ratio.

### 6. Portfolio CLI (Command Center)

- **Goal:** A "power user" feature to showcase technical depth.
- **Task:** Implement a global keyboard listener (e.g., `Ctrl + \``) to open a terminal overlay.
- **Commands:** `cd`, `ls`, `help`, `whoami`, `contact --send`.

## ✅ Success Criteria

- [x] Backend tracking logic does not add >10ms to response times.
- [x] Admin dashboard provides actionable insights into chatbot performance.
